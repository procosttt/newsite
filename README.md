# «По шаблону !» — мини-платформа для подготовки к ЕГЭ по информатике

Flask + Jinja2 + ванильный JS/CSS.  
Данные лежат в JSON.  
Редактор кода — CodeMirror 6 через CDN (если не загрузится, будет textarea).  
Тема и сохранённые решения задач — в localStorage.

## Маршруты
- `/` — Главная
- `/templates` — список номеров (шаблоны)
- `/templates/<id>` — карточка шаблона: объяснение + код + копирование
- `/tasks` — список номеров (задания) + поиск + прогресс (localStorage)
- `/tasks/<id>` — задачи + редактор + сохранение/сброс/копирование
